# nodejs的爬虫

::: tip
实际上爬虫并没有非常的复杂，主要就是请求网站地址，将网站的发回的数据进行数据处理，实现的方法又很多种，无非就是那种方法更加的便捷，当然网站开发者也会想办法来反爬，这就是一场关于双方的博弈。

:::

## 依赖包
- `axios`
> 用于发送网络请求
- `cheerio`
> 将请求回来的数据进行处理，`cheerio`可以像jq一样在弄得中处理网站发来的内容，非常好用 


## 实战
> 我随便找了个电影网站 `https://www.piaohua.com/` 尝试去爬取它的数据

### 安装依赖
```
npm i axios cheerio
```
