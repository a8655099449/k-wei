## 截取整个页面
如果我们要截取整个网页的长图 ，控制台提供了内置的方法，不再需要任何的工具

`打开控制台输入箭头所指的命令即可截取整个网页的长图`

![alt](//image.woai996.com/picgo/20210221160912.png)

如果我们只是需要某个元素的节点，使用鼠标选择改元素后，用控制台选择箭头所指的命令即可生成

![alt](//image.woai996.com/picgo/20210221161351.png)
